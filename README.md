### [analytics-mlops](https://github.com/icesrc/analytics-mlops)
**End-to-end MLOps pipeline for daily KPI forecasting**

- Raw daily metrics → EDA → 3 ML models with time-series cross-validation → production retraining → automated HTML reports
- Two AI agent integrations: one auto-injects insights into reports, the other simulates team responses to proposed system changes
- Dataset-agnostic — swap the CSV and run
